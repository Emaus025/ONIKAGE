Onikage
https://via.placeholder.com/800x200/333333/FFFFFF?text=Onikage

Un videojuego 2D en desarrollo con Unity, inspirado en el estilo y mecánicas de Undertale. Sumérgete en un mundo lleno de misterio, decisiones morales y combates únicos que desafiarán tus principios.

Sobre el Juego
Onikage es una experiencia narrativa 2D que combina:

Combates por turnos con mechanics de bullet hell

Sistema de decisiones que afecta la historia

Personajes memorables y diálogos profundos

Mundo pixel art con atmósfera única

Características
Sistema de combate híbrido: Alterna entre acciones pacíficas y combates intensos

Múltiples finales: Tus decisiones cambian el destino de los personajes

Banda sonora original: Compuesta específicamente para cada momento

Controles precisos: Responsive y adaptado para diferentes estilos de juego

Historia emocional: Un relato que explora temas de redención y identidad
